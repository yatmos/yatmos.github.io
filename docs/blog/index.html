      - 대기/해양 시계열 층별 자료 고급분석 및 맵핑
      - 영문 보고서 작성 및 영어 논문 작성 (reading / writing)
      - 조사선 승선 해양-대기 관측 참여

  3. 자격요건:
      - 학력 : 석사 이상
      - 경력 : 신입 / 경력

  4. 우대사항:
      - 박사학위 혹은 postdoc 우대
      - Matlab / Python 숙련자
      - 대기 / 해양 관측 센서에 대한 이해도가 높은 자

  5. 근무조건:
      - 고용형태 : 정규직
      - 우대사항 : 보훈대상자 채용 평가 시 가점 부여
      - 근무 요일/시간: 주 5일 (월-금) / 출근시간: 8시-10시, 퇴근시간: 17-20시 (유연근무제 적용)
      - 급여조건 : 24년 정규직 기준 학사 초임 3,840만원, 석사 초임 4,390원
       (중식대 포함, 박사학위, 경력 및 조건에 따라 상향 가능)
      - 본사주소 : 경기도 군포시 엘에스로 172, 한림 휴먼타워 306호

  6. 전형절차:
       서류전형 > 온라인 인성검사 > 면접전형 > 패용건강검진 > 최종합격 (* 석사 이상 면접 당일 PT 면접 진행)

  7. 제출서류:
       이력서, 자기소개서, 성적증명서, 졸업증명서, 경력증명서(경력직), 추천서(보유자), 기타 우대사항 증명
       (* 서류전형 합격자는 추후 자사양식 이력서 제출 요청, 경력자는 해당 직장에서 발급받은 경력증명서 제출 필수)

  8. 문의사항: 경영 총괄부 이수민, 031-5180-5731, www.geosr.com
